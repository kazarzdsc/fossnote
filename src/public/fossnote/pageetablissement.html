Fossnote

Serveur PRONOTE auto-hébergé, open source et gratuit

(Free Open Source Selfhostable PRONOTE)

Installation

$ git clone https://github.com/CaraPloof/fossnote
$ cd fossnote
$ npm install
$ npm start

Accédez ensuite à localhost:3000/fossnote/.
